# Softadastra: The C++ Tooling Company

We are building modern, coherent, and powerful tools to make C++ development simpler, faster, and more productive.

Softadastra exists to simplify the way developers create, build, test, diagnose, package, and distribute C++ projects.

Learn more at [softadastra.com](https://softadastra.com)

## Our Focus

Softadastra is focused on one clear niche:

**C++ tooling.**

If a project does not directly help simplify C++ development, it is not a priority for Softadastra.

## Vix.cpp

Vix.cpp is not only a C++ framework.

The goal is to make C++ projects easier to create, build, test, diagnose, package, and distribute.

## Softadastra Engine

Softadastra Engine helps C++ applications keep working when internet access is slow, unstable, expensive, or temporarily unavailable.

It provides an offline-first foundation for:

- Write-ahead logging
- Local state
- Store
- Sync
- Retry
- Delivery
- Transport
