# Softadastra: The C++ Tooling Company

We build and maintain open C++ tooling to make modern C++ development simpler, faster, and more accessible. Learn more at https://softadastra.com/

## Our Projects

- [Vix.cpp](https://github.com/vixcpp/vix) - the modern C++ toolchain foundation
- [Softadastra Engine](https://github.com/softadastra/softadastra) - offline-first runtime layer for C++ applications
- [cnerium](https://github.com/softadastra/cnerium) - a reliability-first backend layer for Vix
- [kordex](https://github.com/softadastra/kordex) - a JavaScript runtime for reliable local-first applications, based on Vix.cpp
